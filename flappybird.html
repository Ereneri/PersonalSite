Flappy Bird

Built a fully working Flappy Bird clone in Java along with a neural network that learns to play the game. The game was built in ONLY 2 days during finals week as a fun quick project. The neural network was built using a genetic algorithm, called NEAT, which stands for NeuroEvolution of Augmenting Topologies. The neural network was trained using a genetic algorithm to play the game. I built the NEAT implementation from scratch following the original white paper and various online resoruces. To check it out, check my github link above. There are two versions of the game, one where the bird is controlled by the player and the other where the bird is controlled by the neural network, the latter is still in-progress.
